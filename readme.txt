Addon that integrates User Switching to Ultimate Member

== Description ==
Adds a link to member profile header section of Ultimate Member for switching to a user.

= Compatibility =
- Tested up to 1.4.0 of User Switching
- Tested up to 2.0.35 of Ultimate Member

== Installation ==

1. Download and extract the zip archive
2. Upload `um-user-switching` folder to `/wp-content/plugins/`
3. Activate the plugin through the \'Plugins\' menu in WordPress

== Frequently Asked Questions ==

= Do I need another plugin for this? =

This plugin requires both [Ultimate Member](https://wordpress.org/plugins/ultimate-member/) and [User Switching](https://wordpress.org/plugins/user-switching/)

= Where can I learn more about User Switching? =

This plugin compliments User Switching, you can find more details about [User Switching here](https://wordpress.org/plugins/user-switching/)

= Where can I find the switch button/link? =

A link is placed inside of the Ultimate Member profile header.

= Can a regular user switch to another user account? =

This plugin uses the same role and access permission as User Switching and will adjust accordingly.

= Does this gives the option to create child accounts? =

No, UM User Switching doesn't grant the ability to create and manage child accounts.

= Is there a Pro version to this? =

No. This plugin is free and open-sourced like Ultimate Member and User Switching
